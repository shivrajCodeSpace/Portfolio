# Portfolio Web Application

This repository contains a modern, production-ready portfolio web application with a Next.js frontend and an Express.js backend.

## Project Structure

- frontend/ - Next.js client application
- backend/ - Express.js API server
- docs/ - Project documentation and architecture notes

## Getting Started

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm run dev
```
