# Architecture Overview

## Frontend

The frontend is a Next.js application using the App Router. It is responsible for rendering the public portfolio pages and future admin views.

## Backend

The backend is an Express.js server that exposes REST-style health and future content APIs.

## Folder Responsibilities

- frontend/ - UI and client-side experience
- backend/ - server-side business logic and API endpoints
- docs/ - architecture, setup, and deployment guides
